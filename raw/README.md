# Raw Inputs

Use this folder locally for raw inputs that an assistant should process.

## Subfolders

- `model_papers/`: public model papers, reports, or citations used for Learn Cards.
- `operational_notes/`: latest model-run notes, data cut-offs, source lags, and implementation details.
- `use_cases/`: use case definitions required before a final checklist can be completed.
- `meeting_notes/`: public-safe notes or redacted summaries for knowledge updates.

## Public Repo Rule

This public repo should keep `raw/` as a convention, not a data dump. Do not commit confidential IPC material, restricted partner input, unpublished operational notes, or private local paths.
