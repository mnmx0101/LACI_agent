# LACI Operational Note Requirement

- **Rule:** Every IPC Analysis Use Checklist requires the latest operational note.
- **Missing Note:** Checklist is preliminary only.
- **Stale Note:** Reassess first.
- **Current Note:** Proceed to context-specific limitation match.

## Required Fields

- **Version / Run Date / Extract Date**
- **Data Cut-Off**
- **Latest Input Dates By Source**
- **Spatial Unit And Boundary Version**
- **Temporal Unit And Prediction Horizon**
- **Current Threshold / Signal Rules**
- **Known Source Lags**
- **Version Changes**
- **Operational Caveats**
