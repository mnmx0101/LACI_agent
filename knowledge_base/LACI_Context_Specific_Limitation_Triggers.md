# LACI Context-Specific Limitation Triggers

## Core Rule

The use checklist must compare known model limitations against the current IPC analysis environment.

## Trigger Labels

- `TRIGGERED - Downgrade`
- `TRIGGERED - Reassess`
- `UNKNOWN - Cannot rely`
- `NOT TRIGGERED - Proceed with caveat`

## Required Comparison

| Known Model Limitation | Current Situation / Analysis Environment | Trigger Status | Decision Implication |
| :--- | :--- | :--- | :--- |
| [Limitation] | [Current condition] | [Trigger] | [Decision effect] |

## Performance Requirement

Every checklist should include the best available performance metric and explain what it supports for the specific use case and what it does not support.
