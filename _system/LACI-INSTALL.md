# LACI_agent Install And First Open

## Clone

```bash
git clone https://github.com/mnmx0101/LACI_agent.git
cd LACI_agent
```

## Open With An Assistant

- **Codex:** read `AGENTS.md`, then `setup/CODEX.md`.
- **Claude:** read `CLAUDE.md`, then `setup/CLAUDE.md`.
- **Cursor or another assistant:** read `README.md`, `ENTRY.md`, `CORTEX.md`, and `TASKS.md`.

## First Local Folders

This repo includes placeholder `raw/` and `outputs/` folders. Use them for local work, but do not commit confidential inputs or private generated outputs.

## Recommended First Test

1. Read `demo/README.md`.
2. Open `demo/yemen_walkthrough.md`.
3. Ask the assistant to explain the Learn -> Assess -> Use Checklist path.
4. Run one draft task using only public-safe example material.
