# Outputs

Use this folder locally for generated LACI artifacts.

## Subfolders

- `learn_cards/`: Learn Card drafts and reviewed versions.
- `assess_cards/`: Assess Card drafts and reviewed versions.
- `use_checklists/`: IPC Analysis Use Checklist drafts and final decision-support versions.
- `reviews/`: red-team reviews, IPC process reviews, and model-builder feedback summaries.
- `knowledge_updates/`: proposed wiki or knowledge-base updates.

## Public Repo Rule

Commit only public-safe examples or placeholders. Keep real operational outputs local unless your team explicitly approves sharing them.
